Machine Learning

In a machine learning course at Shenandoah Valley Governor's School, I created the Break Point Predictor for competitive bowlers. With the donation of an extensive research dataset from the Professional Bowlers Association, I implemented an XGBoost gradient boost algorithm to guess where the optimal location to bowl a strike is.

The Break Point Predictor takes the input of a few key details about a given shot to determine the length of the oil pattern. The point where the oil stops on a boling lane is called the break point because it is where the hooking bowling balls will 'break' towards the pins. The algorithm is able to accurately estimate the break point within a foot from the actual length of the oil.

The goal of this project was to balance accuracy with ease of use for bowlers without the ability to find precise data measurements. With that in mind, the Break Point Predictor went above and beyond what I thought was possible. The success of this project earned the Best in Machine Learning Award from Shenandoah Valley Governor's School.
